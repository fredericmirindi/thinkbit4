Research Focus Areas

Our big data analytics research encompasses distributed systems design, real-time processing frameworks, and advanced data mining techniques for extracting insights from massive datasets.

Distributed Systems

Advanced architectures for horizontally scalable data processing across clusters of commodity hardware, focusing on fault tolerance and data consistency.

Apache Hadoop Ecosystem Apache Spark Optimization Distributed File Systems Fault-Tolerant Computing

Real-Time Analytics

Low-latency data processing systems that enable instant insights from streaming data sources, supporting real-time decision making at scale.

Apache Kafka Streaming Apache Flink Processing Event-Driven Architecture Complex Event Processing

Data Mining & Pattern Discovery

Scalable algorithms for extracting meaningful patterns and knowledge from massive datasets using machine learning and statistical methods.

Clustering Algorithms Association Rule Mining Anomaly Detection Predictive Analytics

Data Lake Architecture

Flexible storage architectures that accommodate structured, semi-structured, and unstructured data with advanced governance and metadata management.

Multi-format Storage Data Governance Schema Evolution Cloud-Native Solutions
