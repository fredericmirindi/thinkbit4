## 🚀 **Getting Started**

### **Quick Setup**

1. **Clone the repository**
   ```bash
   git clone https://github.com/fredericmirindi/thinkbit4.git
   cd thinkbit4
   ```

2. **Open locally**
   ```bash
   # Using Python (Python 3)
   python -m http.server 8000
   
   # Using Node.js
   npx serve .
   
   # Using PHP
   php -S localhost:8000
   ```

3. **Visit the site**
   ```
   http://localhost:8000
   ```

### **Development Setup**

For advanced development with live reload:

```bash
# Install live-server globally
npm install -g live-server

# Run with live reload
live-server --port=8000 --open=/
```

## 🔧 **Customization**

### **Theme Customization**

To customize the theme colors, modify the CSS custom properties in `assets/css/style.css`:

```css
:root {
  --primary-color: #your-color;
  --secondary-color: #your-color;
  --accent-color: #your-color;
}
```
